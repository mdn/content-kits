MDN Content Kits

MDN Content Kits are everything you need to host a local developer meetup or present on a technical topic at an event.

Each MDN Content Kit contains a collection of resources about a particular topic related to web development or web app development, such as Web APIs, useful libraries, or developer tools. Resource may include slides, demos, videos, exercises, etc.

By providing MDN Content Kits, MDN hopes to grow developer engagement with Mozilla at a local level, and increase standards-based web development globally.

MDN Content Kits

  • Beginners Programming

    This kit contains resources that you can use the learn and teach general programming principles and the JavaScript language. The teacher will need to know HTML and JavaScript. The students will not need any previous web development knowledge but knowing how the web works and what web pages are will help.

    The Beginners Programming kit has 8 sections, each of which takes 1-1.5 hours to teach.

    You can try out the Beginners Programming Content Kit now.

  • Valence

    Valence is the public-facing name for the Firefox Developer Tools Adapters, which extend the Firefox WebIDE to allow for remotely debugging pages in Chrome or Safari from within the Firefox Developer Tools. Valence is still in development and is experimental. Valence is distributed as an add-on which is automatically downloaded and installed by Firefox Developer Edition, though the raw .xpi file is available on ftp.mozilla.org and its source code is on GitHub.

    The Valance Content Kit is for an experienced web developer to give a presentation to a group of experienced web developers.

    You can try out the Valence Content Kit now.

How to Propose a New MDN Content Kit

If you would like to propose a new topic for an MDN Content Kit, please add the topics to this etherpad as well as on the MDN mailing list.

If you propose a new Content Kit topic, please let us know how you will use the new MDN Content Kit and whether you will provide us feedback. We are looking for proposals for topics that will be used by groups that can provide us with feedback so that we can continue to improve the MDN Content Kits.

Creating an MDN Content Kit

Kits can be built by anyone, and we encourage people to suggest and build kits! Read about the project on the MDN wiki or get started on the MDN Content Kit Github Project.
